The ascii schematic above is a simple CMOS inverter. It's built with two complementary MOSFETs (NMOS and PMOS) which allows for very efficient power consumption. In digital logic, this is essentially a NOT gate. This is a very simple design; however, it took me a while to draw it out as ASCII.
